# Álbum de Figurinhas da Copa do Mundo

**Aluno:** Lucas Cunha  
**Curso:** Informatica  
**Unidade Curricular:** Codificar aplicações para dispositivos móveis

## Sobre o Projeto

Aplicativo mobile de álbum de figurinhas da Copa do Mundo desenvolvido com Vue 3 e Ionic Framework 

## Funcionalidades

- ✅ Autenticação (login, cadastro, recuperação de senha)
- ✅ Álbum de figurinhas organizado por seleção
- ✅ Marcar figurinhas como coletadas/pendentes
- ✅ Pesquisa por jogador ou seleção
- ✅ Filtros (todas, coletadas, pendentes)
- ✅ Tela de perfil do usuário
- ✅ Feedback visual com toast e badges

## Tecnologias

- Vue 3
- Ionic Framework
- TypeScript
- Vue Router

## Como Executar

```bash
npm install
ionic serve
